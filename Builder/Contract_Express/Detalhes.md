# Tipos de Campos de Formulário Disponíveis

O Contract Express suporte vários tipos de variáveis que determinam como os valores são coletados no questionário Contract Express by Thomson Reuters | Legaltech Hub:

## Campos de Texto:

### Texto simples: Qualquer linha única de texto
### Texto multilinha: Texto de múltiplas linhas
### Texto com formatação rica: Com opções de formatação HTML

## Campos de Seleção:

### Seleção única (dropdown): Lista suspensa com uma opção
### Botões de rádio: Grupo de botões onde só um pode ser selecionado
### Lista de seleção múltipla: Permite escolher nenhuma, uma ou várias opções
### Checkboxes: Grupo de caixas de seleção
### Lista pesquisável: Caixa de texto que filtra opções conforme digitado

## Campos de Data/Hora:

### Calendário: Caixa de texto com botão que abre calendário popup
### Por formato: Mistura de caixas de texto e listas suspensas

## Campos Numéricos:

### Número inteiro: Números inteiros positivos e negativos
### Número decimal: Qualquer número incluindo decimais

## Campos Especiais:

### Sim/Não: Botões de rádio ou checkbox único
### Upload de arquivo: Para carregar documentos
### Entidade: Coleção de dados de múltiplos tipos de variáveis
